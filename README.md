# 🛒 Insight-Cart
### Retail Sales Analytics Dashboard using Power BI

Transforming retail transaction data into actionable business insights through data cleaning, analysis, and interactive visualizations.


---

## 📖 Overview

Insight-Cart is a Business Intelligence project built using **Power BI** to analyze online retail sales data. The dashboard provides interactive visualizations that help understand sales performance, customer behavior, product trends, and country-wise revenue.

This project demonstrates the complete analytics workflow, including:

- Data Cleaning
- Data Transformation
- Exploratory Data Analysis (EDA)
- Interactive Dashboard Development
- Business Insight Generation

---

## 📊 Dashboard Features

- 📈 Sales Performance Overview
- 🌍 Country-wise Revenue Analysis
- 🛍️ Product Performance
- 👥 Customer Purchase Analysis
- 📅 Sales Trend Over Time
- 💰 Revenue Distribution
- 🔍 Interactive Filters & Slicers

---

## 🗂 Dataset

The project uses an Online Retail dataset containing transactional sales data.

### Dataset Columns

| Column | Description |
|---------|-------------|
| InvoiceNo | Invoice Number |
| InvoiceDate | Date of Purchase |
| CustomerID | Customer Identifier |
| Description | Product Name |
| Quantity | Quantity Purchased |
| UnitPrice | Price Per Unit |
| TotalAmount | Total Transaction Value |
| Country | Customer Country |
